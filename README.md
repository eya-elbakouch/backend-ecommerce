# 🐾 Petchologie - Plateforme pour animaux

## 📌 Description

**Petchologie** est une plateforme web dédiée aux animaux de compagnie.
Elle permet aux utilisateurs d’accéder à des informations sur **l’alimentation animale** ainsi que sur les **services vétérinaires**.

L’objectif est d’améliorer le bien-être des animaux en facilitant l’accès aux ressources essentielles pour leurs propriétaires.
